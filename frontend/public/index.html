Understanding the Analysis

Loading Journey

Shows the chronological sequence of how your website loads resources from different global locations.

  • • Pins: Server locations hosting website assets
  • • Lines: Network connections from your location
  • • Timeline: Request/response sequence with timings

Asset Density

Visualizes where the majority of website infrastructure is concentrated globally.

  • • Heat zones: Areas with high server concentration
  • • Color intensity: Brighter = more servers
  • • Hot zones list: Ranked by asset count per location

DNS Performance

Measures how quickly different global DNS servers can resolve the website's domain.

  • • Blue zones: Fast DNS resolution (<50ms)
  • • Red zones: Slower DNS resolution (>200ms)
  • • Latency list: DNS servers ranked by response time

How Data is Obtained

Analysis is performed using real browser automation to load the website and capture network traffic. Geolocation data is obtained through IP address lookup, and DNS latency is measured from 10+ global DNS servers. The visualization shows actual network paths and timings from your approximate location.
